guest@securesystem:~$

Type 'help' to see available commands. Complete the cybersecurity challenges to earn badges.

Available Commands

help - Display available commands
clear - Clear the terminal
ls - List files and directories
cat [file] - Display file contents
scan - Scan system for vulnerabilities
crack [file] - Attempt to crack a password file
decrypt [file] - Decrypt an encrypted file with a key
whoami - Display current user
login [user] - Login as a different user
